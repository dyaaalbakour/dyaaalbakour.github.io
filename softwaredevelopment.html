Expertise in Software Development


I have worked on several software projects where I developed and deployed systems using the a wide range of technologies, most notably:
  • The Terrier IR platform
  • Apache Storm for reliable distributed stream processing
  • Enterprise search with open source SOLR.
  • Restful Web Services.
  • Cloud-based platforms (Microsoft Windows Azure).
  • The UIMA Framework for distributed Text Processing.
An example of a public system I built is the University of Essex search engine. This search engine is used every day by thousands of users and it has an interactive query suggestion feature that implements some of the methods I developed during my PhD.
